Topik Aplikasi:
Aplikasi ini dibuat menggunakan bahasa pemrograman Kotlin dan dirancang untuk memudahkan pengguna dalam melihat, mencari, dan mengurutkan menu sandwich yang tersedia.
Selain menampilkan daftar menu, aplikasi ini juga menerapkan beberapa konsep pemrograman dan algoritma, seperti Linear Search untuk pencarian menu dan Bubble Sort untuk pengurutan data.
